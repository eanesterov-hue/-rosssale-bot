# Cross Sale Bot

Инструмент для кросс-продаж: поиск брокеров, которые вели показы по заданному ЖК или району. Помогает быстро найти, к кому обратиться внутри компании, когда появился объект или собственник.

**Данные.** Вся информация загружается вручную. Файл выгрузки «Показы» из AmoCRM (таблица с колонками Брокер, Дата, Объект) кладётся в `data/showings.xlsx`. Справочник районов — в `data/districts.json`.

**Интерфейсы.** Скрипт работает локально. Есть два способа использования:
- **Telegram-бот** — пишете название ЖК или «район Хамовники», бот возвращает список брокеров за последние 60 дней.
- **CLI** — команда `python scripts/main.py --object "Прайм парк" --days 14` выводит список в терминал.

**Запуск.** Установите зависимости (`pip install -r requirements.txt`), скопируйте `.env.example` в `.env`, укажите `TELEGRAM_BOT_TOKEN`, запустите `python scripts/bot.py` — для бота, или `python scripts/main.py` — для CLI.
